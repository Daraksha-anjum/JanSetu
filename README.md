# 🌍 JanSetu

> AI-powered hyperlocal civic issue reporting platform built using Google Gemini AI, Firebase, and Google Cloud.

---

## 📌 Overview

JanSetu is an intelligent community platform that enables citizens to report civic issues such as potholes, water leakages, broken streetlights, garbage accumulation, and drainage problems. Using Google Gemini Vision, the platform automatically analyzes uploaded images/videos, categorizes issues, estimates severity, and recommends the appropriate municipal authority.

The platform also enables community participation through verification, discussion rooms, AI-generated insights, and coordinator dashboards for transparent issue resolution.

## ⚙️ Getting Started

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Create a `.env` file using `.env.example` and configure:

- Gemini API Key
- Cloudinary Credentials

Firebase configuration is already included in the project.
